05 / Motion

Motion & Animations

Motion defines the fluidity of the user journey. The site uses the Animate On Scroll (AOS) engine, triggering deep entries (1500ms duration, offset delays) as users navigate down sections. Hover states use subtle ease transitions on opacity, font family weightings, and transform translations.

AOS Scroll Animation Simulator

Trigger entrance curves at the exact production spec: 1.2s - 1.5s duration

Live Preview Area

2 Rivers Sandbox

Observe the entry curve. The transition uses ease-in-out properties to ensure premium velocity decay.

Entrance Spec Schema

Property Reference Value Aesthetic Purpose
Duration 1200ms - 1500ms Provides a slow, cinematic reveal.
Easing cubic-bezier(0.25, 0.46, 0.45, 0.94) Decelerates softly at the end.
Offset Distance 40px translation Ensures entries feel light and elegant.
Cascade Delays 300ms, 500ms, 700ms, 900ms Guides eyes from image to titles.

Hover Transition Spec

Image Grayscale to Full Color Transition: filter 0.5s ease. Grayscale is active by default to enforce palette neutrality, shifting to full color on element hover to invite user focus.

Drawer Height Reveal (Property Details) Transition: max-height 0.4s ease-out, opacity 0.3s ease. Starts collapsed at 0px with opacity-0, expanding to reveal amenities list on card hover.

Label Bottom Offset Translation Transition: bottom 0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94). Translates text positions upwards from 28px to 45px on container hover.
